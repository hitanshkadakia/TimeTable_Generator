Time Table Generator for SBMP (Shri Bhagubhai Mafatlal Polytechnic)

Time table generation is tedious job for educationalist with respect to time and man power. Providing an automatic time table generator will help to generate time table automatically. Proposed system of our project will help to generate it automatically as well as helps to save time. It avoids the complexity of setting and managing Timetable manually.

These algorithms incorporate a numeral of strategy, aimed to improve the working of the search operation. The system will take various inputs like number of subjects, teachers, and workload of a teacher, semester, and priority of subject, classroom & laboratory. By relying on these inputs, it will generate possible time tables for working days of the week for teaching faculty. This will integrate by making optimal use of all resources in a way that will best suit the constraints. Basically our project is a web application developed using the Django Framework and a SQLite database.
